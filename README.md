# mini_os

mini_os

A minimal operating system for learning and experimentation.

🚀 About

mini_os is a minimal OS project focused on understanding low-level programming and operating system development.

This project is designed as a minimal operating system, meaning it only includes essential functionality without unnecessary complexity.

The long-term goal is to orient the system toward drone applications, where a full-featured OS is not required. Instead, the focus is on:

high performance
low latency
reliability
memory safety
